# AI 辅导网站实现计划

## 项目概述
- **目标**：创建 AI 辅导网站，支持 Web 和 Android 手表端
- **部署**：Vercel
- **核心功能**：图片上传到 AI（AI 已支持图片识别）

## 任务列表

### 1. 环境变量配置
- 创建 `.env.local` 文件（本地开发）
- 创建 `.env.example` 文件（模板）
- 配置 `NEXT_PUBLIC_AI_API_URL` 和 `NEXT_PUBLIC_AI_BASE_URL`

### 2. 响应式布局设计
- 设计移动端优先的布局
- Android 手表端适配（小屏幕优化）
- Tailwind CSS 断点配置

### 3. 图片上传功能
- 图片选择组件
- 图片预览功能
- 图片压缩/优化
- 上传状态反馈

### 4. AI API 集成
- API 客户端封装
- 图片上传到 AI
- 响应处理
- 错误处理

### 5. Vercel 部署配置
- `vercel.json` 配置
- 环境变量设置说明

## 技术栈
- Next.js 16 + TypeScript
- Tailwind CSS v4
- Vercel 部署
